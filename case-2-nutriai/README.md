# Case 2 — NutriAI: карточка нутриентов по результатам обследования

> Этот кейс — часть [тестового задания](../README.md). Общий отчёт пиши в `../REPORT.md`.

## Контекст

NutriAI — наш стартап нутрициологии. Идея: пользователь загружает результаты медицинского обследования — система выдаёт **информационную карточку** про каждый маркер: что это за показатель, что с ним сейчас (норма / отклонение), что это значит, общие источники этих веществ.

> **Важно:** NutriAI **не ставит диагнозы и не назначает лечение**. Это информационный продукт. Все формулировки должны это подчёркивать («рекомендуем обсудить с врачом», «консультация специалиста», и т.д.). Никаких «принимайте N мг витамина X» — это медицинский совет, и его мы не даём.

## Задача

В `data/blood_test_2026-04-15.txt` лежит обезличенный результат биохимического анализа крови. Тебе нужно превратить его в **markdown-карточку**, которая войдёт в базу знаний NutriAI.

1. **Разбери результаты** — какие маркеры есть, какие значения, какие референсы (нормы) указаны.
2. **Определи отклонения** — какие показатели вышли за норму, в какую сторону.
3. **Напиши markdown-карточку** в файл `output/card.md` со структурой:
   - Дата / тип обследования
   - Сводка одной фразой: что в норме, что нет
   - **Таблица**: маркер, значение, единица, референс, статус (норма / ниже / выше)
   - На каждое отклонение — короткий блок (3-5 фраз):
     - что это за вещество / маркер
     - в общих чертах, что значит этот уровень
     - природные пищевые источники этого вещества
     - **обязательно**: «обсудите с врачом / нутрициологом»
   - **Disclaimer внизу**: «Это информация общего характера, не медицинский совет, не диагноз».
4. **Источники** — для каждого утверждения о норме/функции вещества **обязательно** указывай источник (ссылка на статью, гайдлайн, книгу). Без источников = брак.

В `example_done/card_example.md` — пример того, как может выглядеть готовая карточка для **другого** обследования. Это образец формата, не копируй текст напрямую.

## Где могут быть подводные камни

- В результатах могут быть **маркеры с непривычными названиями** (сокращения, латынь). Гугли, разбирайся.
- **Референсные диапазоны различаются** между лабораториями — используй те, что в самом анализе.
- Часть значений может быть на **границе нормы** — описывай как «норма, но близко к границе», не делай вид что всё идеально.
- **Не выдумывай функции** маркеров — если не уверен, признайся в карточке («роль этого маркера сложная, изложение упрощённое»).

## Что мы оцениваем

- **Аккуратность работы с медицинскими формулировками** — без диагнозов, без советов о приёме веществ
- **Источники** — реальные, проверенные, по делу. Никаких ссылок на «sciencedirect.com» без конкретики
- **Структура карточки** — следуешь образцу или делаешь по-своему обоснованно
- **Внимательность** — заметил все отклонения, не упустил скрытые

## Сколько времени

~1 час.

## Куда складывать результат

```
output/
  card.md          ← твоя карточка
  sources.md       ← список использованных источников (опционально, можно прямо в card.md)
```
