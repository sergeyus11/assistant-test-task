# Тестовое задание — AI-помощник для CEO с 10+ проектами

Привет! Если ты дошёл до этого репо — спасибо за интерес к вакансии. Дальше — тестовое.

## О чём вакансия

Мы ищем человека, который **через AI-инструменты** (Claude Code, ChatGPT, Cursor) поможет нам управлять портфелем из ~10 разных проектов: кондитерское производство, аналитика, медицинский стартап, блокчейн-сеть, контент-блог, и т.д. Ты не пишешь код с нуля, ты используешь AI чтобы разбираться в данных, чистить таблицы, заводить контент, проверять расчёты, готовить документы.

Главное качество, которое мы проверяем этим тестом — **способность сам разобраться** в незнакомом домене. Мы НЕ проверяем знание кондитерки, медицины или блокчейна. Мы проверяем **подход**.

## Что в тестовом

3 разноплановых кейса, имитирующих типичный день AI-помощника. Каждый — отдельная папка:

```
case-1-crystalfactory/    Кондитерка: разобрать данные из 1С, привести к формату
case-2-nutriai/           Контент: разобрать результаты обследования, написать карточку
case-3-gonka/             Tech docs: по коммитам в git собрать changelog
REPORT_TEMPLATE.md        Шаблон твоего общего отчёта по всем 3 кейсам
```

В каждой папке свой `README.md` с задачей, `data/` с входными файлами, `example_done/` с одним готовым примером (где применимо).

## Сколько времени

**Реальная работа — 3-5 часов.** Дедлайн 5 дней с момента получения. Если что-то занимает дольше 1.5-2 часов на один кейс — не страшно, опиши в отчёте.

## Что прислать

1. **Свой git-патч** или ссылка на форк/гист с твоей работой (форкни этот репо или сделай свой публичный)
2. **`REPORT.md`** — общий отчёт: что сделал, что заметил, какие приняты решения, какие вопросы остались (по каждому кейсу — отдельная секция)
3. **`prompts.md`** — главные промпты, которые ты давал AI (можно по 2-3 ключевых на каждый кейс)
4. **Скринкаст 10-15 минут** (Loom / OBS / любой) — покажи **процесс**, как ты разбираешься, не только результат

## Что мы оцениваем

| Критерий | Что смотрим |
|---|---|
| Подхват контекста | Разобрался в незнакомом домене или нет? Сделал по образцу или механически? |
| Заметил странности | В каждом кейсе намеренно есть «ловушки» (не злонамеренные — реальные проблемы данных). Заметил? Описал? |
| AI-промпты | Используешь AI как инструмент или как поисковик? Итеративный процесс или копипаста? |
| Аккуратность с данными | Цифры сходятся? Единицы соответствуют? Проверил себя? |
| Принятие решений | Где данные противоречивы — принял решение и обосновал, или проигнорировал? |
| Отчёт | По REPORT.md видно что ты сделал и думаешь, или это формальная отписка? |
| Скринкаст | Видно как работаешь — мышление вслух, проверка результата AI, не паника при ошибках |

## Что НЕ оцениваем

- Скорость (3 часа или 6 — неважно)
- Красоту кода / красоту markdown
- Идеальный русский
- Знание Django / React / любого фреймворка
- Экспертизу в конкретном домене (кондитерка, медицина, блокчейн)

## Если что-то непонятно

**Принимай решение сам и обоснуй в REPORT.md.** В реальной работе тебе часто будет нужно работать с неполной информацией — это нормально. Если **точно** не можешь без уточнения — открой issue в этом репо с вопросом, мы ответим.
